# 📚 Sistema de Notas Escolares

Sistema web desenvolvido com HTML, CSS e JavaScript puro para gerenciamento de notas escolares.

## 🚀 Funcionalidades

- 🔐 Login com controle de sessão
- 📚 Cadastro de disciplinas
- 📝 Lançamento de notas
- 📊 Cálculo automático de média
- 📈 Gráfico de desempenho (Chart.js)
- 📄 Geração de boletim em PDF
- 🌙 Modo escuro (dark mode)
- 💾 Persistência com LocalStorage

## 🎯 Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js
- jsPDF

## 📌 Objetivo

Projeto com foco educacional e evolução progressiva, simulando um sistema real sem uso de frameworks.
